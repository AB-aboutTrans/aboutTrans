# aboutTrans

关于跨性别，你想知道的都在这里～

这是一个跨性别相关知识科普站，无论是跨儿伙伴还是友跨朋友们都可以在此获取到有用的信息！

## 🌐 访问网站

https://aboutrans.info

## 📑 内容目录

- [概念术语](/docs/terms.md) - 汇总并详解与跨性别相关的名词及专业术语
- [医疗照护](/docs/medical.md) - 提供从精神科诊断到性别肯定激素与手术治疗的相关信息
- [政策法规](/docs/policy.md) - 了解身份信息及学籍相关信息变更的具体方式与要求
- [社会支持](/docs/support.md) - 了解社会各界为支持跨性别群体所做的努力
- [纪念活动](/docs/events.md) - 介绍与跨性别群体有关的国际节日和纪念日
- [议题问答](/docs/faq.md) - 关注国内跨性别议题中常见的问题和误解

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览构建结果
npm run docs:preview
```
