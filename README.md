# aboutTrans

aboutTrans 是一个专注于跨性别与多元性别知识科普的公益性网站，无论是社群伙伴还是盟友朋友们都可以在此获取有用的信息！阅读项目文档，请访问 [aboutrans.info](https://aboutrans.info)。

## 本地预览项目

> 需要先安装 Git、Node.js 与 npm。

```bash
# 克隆项目仓库
git clone https://github.com/AB-aboutTrans/aboutTrans.git
cd aboutTrans

# 安装项目依赖
npm install

# 启动本地预览
npm run docs:dev
```
