## 🚀 本地部署

1. 克隆仓库

```
git clone https://github.com/AB-aboutTrans/aboutTrans.git
```

2. 安装依赖

```
cd aboutTrans
npm i
```

3. 启动开发服务器

```
npm run docs:dev
```
